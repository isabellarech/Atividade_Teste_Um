# Atividade CRUD 01 - Introdução

## Sistema de Login Simples em PHP

## Objetivo da Aplicação
O objetivo da aplicação é desenvolver um sistema simples de login e cadastro de usuários utilizando PHP e MySQL, permitindo a autenticação, além do cadastro e visualização de usuários registrados.

## Tecnologias Utilizadas
- PHP
- HTML
- MySQL
- XAMPP

## Estrutura Básica dos Arquivos

### `index.php`
É a tela de login e validação do usuário.

### `home.php`
É a página principal junto com um formulário de cadastro e exibição dos usuários.

### `tabela.php`
Serve para exibir os usuários cadastrados em uma tabela.

### `conexao_db.php`
Realiza a conexão com o banco de dados.

### `logout.php`
Encerra a sessão do usuário.

## Funcionamento do Código
O usuário realiza o login e depois da autenticação, ele é direcionado para a página principal. 
Já na página principal, o usuário pode tanto cadastrar novos usuários quanto visualizar os registros armazenados no banco de dados. 
O sistema também permite encerrar a sessão por meio do logout (sair).

## Principais Aprendizados Obtidos
- Uso de formulários HTML;
- Integração entre PHP e MySQL;
- Utilização de comandos SQL;
- Controle de acesso com sessões;
- Organização do código em arquivos separados.
